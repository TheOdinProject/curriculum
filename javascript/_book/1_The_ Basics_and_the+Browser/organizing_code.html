Style Guides

Javascript lets you do so many different things that it's easy to feel like you have no organizational structure. Check out these style guides from industry leading companies to improve your stylistic instincts.

  1. Read Organizing your Code from learn.jquery.com
  2. Browse through AirBnB's JS Style Guide is very readable and helpful
  3. If you want some more examples of style guides, check out Airbnb's JS Style Guide, Github's JS Style Guide, and the Contributing to jQuery Style Guide

Other Helpful Stuff

Consider merely browsing the following resources instead of reading with a fine-tooth comb. The idea is to get the concepts and just know that you can always come back when you'd like to know more.

  1. Read Local Storage and How to Use It On Websites from Smashing Magazine to get introduced to a powerful new resource browsers are offering.
  2. Browse the Underscore.js Documentation to learn about a library that makes your Javascript a lot more Ruby-ish.
  3. Browse "Thinking Inside the Box with Vanilla Javascript" from Smashing Magazine to learn a few helpful JS functions
  4. Glance over Working with Regular Expressions in Javascript to see how your Regex knowledge transfers over.
  5. Read Speeding up Page Load Times in Rails by Jarmo Pertman to see ways you lighten your page loads.
  6. Watch Principles of Security by Douglas Crockford (video) for a security primer
